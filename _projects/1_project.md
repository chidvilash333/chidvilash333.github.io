A production-grade **Retrieval-Augmented Generation (RAG)** application built across a 16-file Python project.

**Key Features:**
- Hybrid retrieval combining BM25 sparse search with dense vector embeddings
- Cross-encoder reranking for improved answer precision
- End-to-end RAGAS evaluation (faithfulness, context recall, answer relevancy)
- Modular architecture supporting multiple document formats

**Tech Stack:** Python, LangChain, FAISS, HuggingFace, RAGAS

This project demonstrates production-level thinking in LLM application design — not just building a RAG pipeline, but rigorously evaluating its performance using established metrics.
