A **dynamic programming-based spell correction system** built for CS 5800 (Algorithms) at Northeastern University, featuring an interactive Streamlit demo.

**Key Features:**
- Edit distance (Levenshtein) algorithm for generating spelling suggestions
- Interactive DP table visualizer showing step-by-step computation
- Streamlit web interface for live demonstrations
- Full presentation materials and demo scripts

**Tech Stack:** Python, Streamlit, Dynamic Programming, NLP

This project bridges classical algorithms with modern NLP — visualizing how edit distance underlies real-world spell checkers used in search engines and text editors.
