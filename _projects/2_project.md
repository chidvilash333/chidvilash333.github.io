A **Flask-based web application** that provides AI-powered salary optimization advice tailored to role, experience, and location.

**Key Features:**
- Personalized salary insights powered by the Groq API (LLaMA-based LLM)
- Dynamic prompt engineering based on user inputs
- Clean, responsive web interface
- Deployed and published as an MS portfolio piece

**Tech Stack:** Python, Flask, Groq API, HTML/CSS, JavaScript

This project showcases the ability to rapidly build and deploy LLM-powered web applications — combining backend engineering with practical prompt design.
